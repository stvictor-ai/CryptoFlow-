# CryptoFlow · 币账通

**个人加密货币出入金管理工具** — 记录每一笔资金流动，追踪真实盈亏，分析你的入金行为模式，在 FOMO 冲动时给你一个冷静提醒。单文件部署，无需后端，数据存本地，开箱即用。

## 为什么做这个？

玩加密的人或多或少都经历过这样的循环：**亏了 → 不甘心 → 再充钱 → 又亏了 → 继续充**。

很多人到最后都不知道自己到底往里面扔了多少钱，只有一个模糊的感觉："好像亏了不少"。而这种模糊感恰恰是最危险的——因为你不知道具体数字，就不会真正警觉。

CryptoFlow 就是为了解决这个问题：**把每一笔出入金记下来，用数据告诉你真实情况，在你冲动的时候拦你一下。**

不是交易工具，不是行情软件，就是一个**记账本 + 心理医生**。

## 功能

### 📝 记录管理
- 出入金记录增删改查，支持内联编辑
- 按月分组折叠，不再被几十条记录淹没
- Excel / JSON 导入导出，数据随时带走

### 📊 统计分析
- 总入金、总出金、净盈亏、回本比例一目了然
- 月度/周/年度盈亏统计 + 累计盈亏
- 交易所维度统计
- 月度趋势折线图（入金、出金、净盈亏、累计）

### 🧠 行为分析 & 心理画像
- 分析你的入金行为模式：星期分布、金额区间、入金间隔
- 自动识别危险信号：追单倾向、大额冲动、加码回本、连续入金
- 生成心理标签和针对性建议

### ⚠️ FOMO 防控
- **冷静提醒**：入金时自动检测危险模式（间隔太短、本周太频繁、超出预算、金额异常），弹窗让你停 3 秒想一想
- **月度预算**：设定每月入金上限，进度条实时显示，快到上限变红
- **入金理由**：每次入金强制你写下理由——"为什么要入这笔？"，让你正视自己的动机

### 🎨 体验
- 白天 / 夜间模式切换，自动跟随系统
- 桌面端 + 平板 + 手机端全适配
- 手机端卡片式列表、底部 Tab 栏、浮动添加按钮
- PWA 支持：添加到主屏幕，像 App 一样使用，离线可用
- 80 条交易名人语录，每次打开随机提醒

### 💾 数据安全
- 数据存在浏览器 localStorage，不上传任何服务器
- 自动快照（最多 10 份）+ 手动快照 + 一键恢复
- 可选云同步（自建服务器）

## 使用方式

### 最简单：直接打开
下载 `index.html`，浏览器打开即可使用。无需安装，无需服务器。

### 在线访问
部署在 GitHub Pages：**[https://你的用户名.github.io/CryptoFlow/]()**

### 手机使用
1. 手机浏览器打开上面的链接
2. Safari 点「分享 → 添加到主屏幕」/ Chrome 点「添加到主屏幕」
3. 从主屏幕打开，全屏使用，和原生 App 一样

## 技术栈

单文件 `index.html`，无需构建工具，直接部署。

- Vue 3（CDN）
- Chart.js（CDN）
- SheetJS/xlsx（CDN）
- Day.js（CDN）
- PWA（Service Worker + Manifest）
