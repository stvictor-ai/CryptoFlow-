# 币账通 · 出入金管理 (CryptoFlow)

## 项目概述
个人用的币圈出入金管理工具，支持记录管理、盈亏统计、复盘建议、数据备份与云同步。单文件 HTML 应用，可静态部署。

## 技术栈
- **单文件 `index.html`**（约 1000 行），无需构建工具
- Vue 3（CDN）、SheetJS/xlsx（CDN）、Day.js（CDN）
- 数据存储：浏览器 localStorage
- 云同步：Basic Auth + Express server（`deploy.sh` 含服务端代码）
- 深色金融风 UI，桌面端宽布局 + 手机端底部 Tab

## 文件结构
```
CryptoFlow/
├── index.html      # 主应用（前端全部代码）
├── deploy.sh       # 阿里云一键部署脚本（含 nginx 配置 + Node.js 同步服务）
├── CLAUDE.md       # 本文件
```

### 核心功能
- [x] 出入金记录 CRUD（表格内直接编辑，无需弹窗）
- [x] 快速录入（复制上一笔、交易所下拉补全、输入校验）
- [x] 盈亏规则：出金总额 > 入金总额 → 盈利

### 响应式布局

- [x] 平板（680-860px）：单栏宽表格
- [x] 手机端（<680px）：底部 Tab 切换 + 浮动添加按钮 + 弹出式 Modal

## 部署方式
- **本地使用**：直接浏览器打开 index.html
- **服务器部署**：运行 deploy.sh（需 nginx + Node.js + pm2）
- **Netlify**：直接拖拽 index.html 部署

## 数据模型
```javascript
// localStorage key: crypto_cashflow_entries_v1
[{
  id,           // UUID
  date,         // "YYYY-MM-DD"
  amount,       // 数字，元
  rate,         // 汇率，数字或空
  exchange,     // 交易所名称
  note,         // 备注
  type          // "入金" | "出金"
}]
```
